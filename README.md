# zac.ong

-   **Framework**: [Next.js](https://nextjs.org/)
-   **Deployment**: [Vercel](https://vercel.com)
-   **Content**: [ContentLayer](https://contentlayer.dev/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)

## Running Locally

```bash
$ git clone https://github.com/Trixzyy/zac.ong.git
$ cd zac.ong
$ bun install
$ bun dev
```
